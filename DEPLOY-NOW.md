# Deploy EquipFix AI to Vercel (2 Minutes)

## Step-by-Step

### 1. Go to Vercel
Open: https://vercel.com/signup

### 2. Sign Up with GitHub
- Click "Continue with GitHub"
- Authorize Vercel to access your GitHub account
- Select "Hobby" plan (it's FREE forever for personal projects)

### 3. Import the Project
- Click "Add New..." → "Project"
- Find `arlo-e-dev/equipfix-ai` in the list
- Click "Import"

### 4. Deploy
- Leave all settings as default (Vercel auto-detects Next.js)
- Click "Deploy"
- Wait ~2 minutes while it builds

### 5. Get Your Live URL
- Once deployed, you'll see: `✓ Production: equipfix-ai-xxxxx.vercel.app`
- That's your live site!
- Click the URL to view it

## That's It!

Your site is now live and will auto-deploy whenever you push to GitHub.

## Optional: Add Custom Domain Later

1. Buy domain (Namecheap, GoDaddy, etc.)
2. In Vercel project settings → Domains
3. Add your domain
4. Update DNS records as instructed
5. SSL certificate auto-generated
