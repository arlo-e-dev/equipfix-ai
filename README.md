## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

### Vercel (Recommended)
1. Push to GitHub
2. Import project in Vercel
3. Deploy

## Customization

### Colors
Edit `tailwind.config.ts` to customize the color scheme.

### Content
Main content is in `app/page.tsx`. Update text, pricing, and features there.

### Images
Add images to `/public` directory and reference them in the components.
